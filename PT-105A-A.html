PT-105A

Pressure Gauge

Multi-link Air Leak Tester with a color touch-screen LCD. Up to 8 sensor units can be connected to a single Controller for leak testing.

Wiring for the Controller and the sensor units can be reduced by fieldbus.

Specs

Range: -100 kPa to 700 kPa

source:100 to 240 VAC, 50/60 Hz, 50 VA max

Operating temperature: 5 to 45℃

External dimensions: Controller: W320 x D175 x H302 (excluding protrusions) Pneumatic Circuit Unit: W120 x D301 x H24 (excluding protrusions)

Weight: Controller: Approx. 7.5 kg Pneumatic Circuit Unit: 7.0 kg


Features

Highly sensitive and pressure-resistant DPS. Air operated valves that are not affected by heat.

The Intelligent Pneumatic Circuit with excellent performance and safety checks valve operation and sensor sensitivity in every test.

Master Preset and NR Mode can improve detection and reduce test time.

Extensive channel functions for a wide variety of works.

Functions

Intelligent Pneumatic Circuit

High-performance differential pressure sensor

USB memory data storage

Leak calibratrion

High-pressure compatible
